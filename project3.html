AI powerd dustbin




This project is a robotics system developed using IOT It can detect obstacles and perform automated actions.

Technologies: ESP32 / 4 Ultrasonic Sensors / 2DC motors/




Description:




This project is an IoT-based smart dustbin developed using an ESP32 microcontroller. It is designed to automatically detect nearby objects using ultrasonic sensors and open the lid without physical contact. The system improves hygiene and convenience by reducing the need for manual interaction. It can also be enhanced with IoT features for monitoring and smart waste management. The dustbin uses embedded systems and DC motors to perform automated actions efficiently.
